Why was Babel inevitable?
Aspects to focus on
\begin{itemize}
 \item communication  (eg Erlang)
 \item avoiding common bugs (eg Rust)
 \item avoiding more subtle bugs (eg SML)
 \item handling low level machine features (eg C)
 \item memory constrained situations (eg Forth)
 \item text data (eg PERL)
 \item typography (eg TeX)
 \item boosting an author's ego (eg Aldor) [ha ha ha]
 \item expressing proofs (eg Lean)
\end{itemize}
 No doubt plus a bunch more!
